Relay Vaults Backend is a GraphQL API that allows developers to query vault data. It provides a simple and efficient way to access information about Relay Vaults, including their balances, yields, and other relevant metrics.

The Backend API is accessible at [https://vaults-api.relay.link/](https://vaults-api.relay.link/). It is designed to be easy to use and provides a wide range of queries to retrieve information about the vaults.
