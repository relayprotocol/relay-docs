## Parameters

| Parameter    | Description                                                                                                                                    | Required |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------|----------|
| baseApiUrl       | Base api url for the relay api, defaults to https://api.relay.link but can also be configured to https://api.testnets.relay.link | ❌       |
| options      | Query parameters that map directly to the [currencies api](/references/api/get-currencies)                                                                                                   | ❌       |
| queryOptions | Tanstack query options. Refer to the [Tanstack](https://tanstack.com/query/latest/docs/framework/react/guides/query-options) docs.              | ❌       |

## Return Data

The hooks return an object with the base [Tanstack Query response](https://tanstack.com/query/v5/docs/framework/react/reference/useQuery). The data property maps to the object returned in the aforementioned [currencies api](/references/api/get-currencies).

## Usage

<CodeGroup>
```typescript
import { useTokenList } from '@reservoir0x/relay-kit-hooks'

const { data: suggestedTokens } = useTokenList(
  "https://api.relay.link", 
  { 
    limit: 20,
    term: "usdc"
  }
)
```
</CodeGroup>

## Query Function

```typescript
import { queryTokenList } from '@reservoir0x/relay-kit-hooks'

queryTokenList(
  "https://api.relay.link", 
  { 
    limit: 20,
    term: "usdc"
  }
)

```
