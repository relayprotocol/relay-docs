<Warning>
  `GET /requests/v2` is deprecated. New integrations should use [`GET /requests/v3`](/references/api/get-requests), which requires authentication and offers a richer filter surface and a cleaner response. See the [migration guide](/references/api/api_guides/migrating-to-requests-v3) for what changed and how to upgrade.
</Warning>
