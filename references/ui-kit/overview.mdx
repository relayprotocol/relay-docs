### iFrame Integration

For security reasons, we require all iFrame integrators to be placed on an allowlist. Please contact <a href='mailto:support@relay.link' target="_blank">suport@relay.link</a> to get your domain on the allowlist. 


#### Adding an iFrame to your site
```
 <iframe
  src="https://relay.link/iframe/swap"
  height="100%"
  width="100%"
/>
```
